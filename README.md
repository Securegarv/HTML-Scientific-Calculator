# HTML-Scientific-Calculator

This is a basic scientific calculator web application built using HTML, CSS, and embedded JavaScript. It provides essential arithmetic operations as well as commonly used scientific functions such as square root and trigonometric calculations.

## Features

- Arithmetic operations: Addition, subtraction, multiplication, and division.
- Scientific functions: Square root, trigonometric functions (sin, cos, tan), and exponentiation.
- Clear button: Reset the calculator for new calculations.
- Responsive design: Compatible with various screen sizes for seamless user experience.

## Usage

To use the calculator, simply clone this repository to your local machine and open the `index.html` file in a web browser. Then, you can start performing calculations using the provided buttons.
